# Tic-Tac-Toe Game (US)

This project is a classic Tic-Tac-Toe game with support for both single-player and multiplayer modes, customizable skins, and sound effects.

## üöÄ Features
- üéÆ **Game Modes:** Player vs. Player / Player vs. Bot
- üé® **Customizable Skins:** Various board display styles
- üîä **Sound Effects:** Sounds for moves and game events
- ü§ñ **AI Opponent:** A bot with artificial intelligence to play against
- üé≠ **Interface Color Selection:** Different color schemes for customization

## üîß Installation
1. Make sure you have **Python 3.x** installed.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the game:
   ```bash
   python Game.py
   ```
   **Important!** If there's no sound, run the game from the terminal in the directory containing `Game.py`.

## üéÆ Controls
- **Select game mode:** Enter `1` (PvP) or `2` (play against the bot)
- **Choose interface color:** Enter a color number (1-7)
- **Select board skin:** Enter a skin number (0-8)
- **Choose a cell:** Enter a cell number (1-9)
